Sobre

O Concurso

Incentivando a tecnologia e a sustentabilidade no campo

Para quem é?

Estudantes do Ensino Fundamental e Médio da rede pública do Paraná, matriculados nos componentes de programação e robótica.

Subcategoria 3

Front-End com HTML, CSS e JavaScript. Desenvolva um site completo sobre o tema sustentabilidade no agro.

Prazo Final

14 de junho de 2026 - envio do link do projeto via Ambiente Virtual de Aprendizagem.

Regulamento

Regras Essenciais

Front-End - Subcategoria 3

Tema obrigatório

"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"

Tecnologias permitidas

Apenas HTML, CSS e JavaScript puro (sem frameworks, sem inline CSS/JS)

Publicação no GitHub

Repositório público com tag "agrinho" + GitHub Pages funcional

Organização do código

Código identado, comentários, README.md com instruções e objetivo

Originalidade

Conteúdo autoral ou com créditos para imagens/IA. Proibido cópia de projetos existentes

Não serão aceitos:

  • ❌ Frameworks (React, Vue, Bootstrap)
  • ❌ Programação inline ou interna (CSS/JS no HTML)
  • ❌ Merge de repositórios
  • ❌ Projetos sem responsividade
  • ❌ Falta da tag "agrinho" no repositório
